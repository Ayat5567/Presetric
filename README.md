# Editor Font Pack

This isn’t just a random collection of fonts.

It’s a curated set I use for real edits — cinematic, sigma, clean UI, and branding work.  
Everything here is organized in a way that actually helps you *choose and use* fonts faster, not just scroll through them.

---

## What this repo is about

Most font packs are messy. You download them, forget what’s good, and end up using the same 2 fonts anyway.

This repo fixes that.

- Fonts are grouped by how they’re actually used
- Each font has a small guide (not just a file dump)
- Pairings are included so you don’t guess every time

---

## What’s included

- Clean sans-serif fonts for UI and subtitles  
- Serif fonts for cinematic edits  
- Display fonts for bold/sigma styles  
- A few stylized fonts (glitch, handwritten, anime)

Each font comes with:
- A preview image  
- A short usage note  
- Suggested pairings  

---

## What’s NOT included

You won’t find paid fonts here.

That means:
- No Avenir Next  
- No Arial  
- No “free download” premium leaks  

If a font is licensed, I’ll just reference it instead of uploading it.  
Keeps the repo safe and long-term.

---

## How I use this (simple workflow)

1. Pick a style (cinematic / sigma / clean)
2. Choose a primary font from that category
3. Pair it with a neutral secondary (usually Alte Haas)
4. Keep it minimal

That’s it. No overthinking.

---

## Why this exists

I wanted something consistent.

Not a new style every edit.  
Not random fonts every time.

Just a system I can reuse and improve over time — and other editors can build on too.
